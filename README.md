# AG-Rotas-Urbanas
Serviços urbanos que dependem de deslocamento físico, como saúde comunitária, coleta, manutenção e fiscalização, exigem planejamento eficiente de rotas para otimização de tempo e recursos públicos.
A tecnologia computacional surge como aliada estratégica, permitindo modelar, simular e otimizar percursos por meio de algoritmos especializados.
Apesar do avanço das técnicas de otimização, o uso de algoritmos evolutivos, como os Algoritmos Genéticos, ainda apresenta aplicação incipiente em cenários urbanos simulados voltados à organização de serviços públicos locais, especialmente quando analisado sob uma perspectiva experimental comparativa.
Este trabalho tem como objetivo implementar e comparar diferentes variações de um Algoritmo Genético aplicadas à otimização de rotas urbanas, avaliando seu desempenho em relação a uma abordagem heurística determinística.
Para isso, será construído um ambiente experimental baseado em matrizes de distâncias representando deslocamentos urbanos simulados, no qual diferentes configurações do algoritmo serão executadas e analisadas estatisticamente quanto à qualidade das soluções e estabilidade dos resultados.
Este projeto contribui tanto para a compreensão teórica do comportamento de estratégias evolutivas em problemas de roteamento quanto para a demonstração de sua aplicabilidade prática em cenários urbanos.
